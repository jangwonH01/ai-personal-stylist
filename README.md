# AI Personal Stylist (React)

사진 + 프로필 기반으로 코디를 추천하는 MVP 웹앱입니다.

## 기능
- 정면/측면 사진 업로드 + 미리보기
- 체형 프로필 입력
  - 스타일 기준(여성/남성/유니섹스)
  - 키/몸무게
  - 체형 고민(복수선택)
  - 선호 핏
  - 퍼스널 톤
  - 상황/예산
- 추천 코디 3안 + 이유/팁
- 신뢰도 안내(사진 2장/1장/없음)
- localStorage 자동 저장/복원
